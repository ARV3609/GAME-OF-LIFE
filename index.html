How to Play

1. Add cells to the graph / world
2. Add the cells in clusters as shown below (The image below is just an example you can make your own designs)

3. Click the Play button start the simulation

4. Control the simulation using the controls

5. If you are not able to add cells just click the reset button

6. Finally , sit back and enjoy the simulation
